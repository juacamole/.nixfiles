# Contributing

## Forks

Forks are very much welcomed! This config is meant to be a starting point or reference — take it, break it, make it yours. No need to ask.

## Pull Requests

Please don't open PRs to improve this config. This is a personal setup tailored to my machine and workflow, so I'm not looking for changes to the configuration itself.

## Issues

Issues are welcome though! If you spot a bug, a broken input, or something that looks wrong, feel free to open one. Same if you're using this as a reference and something isn't working as documented.
